rew first

This page contains benchmarks for rew first command.

See benchmark setup for information about testing environemt and input data files.

Benchmark #1

Command:

hyperfine \
    --warmup 5 \
    --setup "rew loop 100 <svejk.txt >input.txt" \
    --cleanup "rm input.txt" \
    "head -n 200000 <input.txt" \
    "coreutils head -n 200000 <input.txt" \
    "rew first 200000 <input.txt"

Results:
