rew lower

This page contains benchmarks for rew lower command.

See benchmark setup for information about testing environemt and input data files.

Command:

hyperfine \
    --warmup 5 \
    --setup "rew loop 1 <svejk.txt >input.txt" \
    --cleanup "rm input.txt" \
    "rew lower <input.txt" \
    "sed 's/\(.*\)/\L\1/' <input.txt" \
    "while IFS= read -r l;do printf '%s\n' \"${l,,}\";done <input.txt"
